# Garage Organization

Affiliate-focused website project for practical garage storage and organization solutions.

## Project Goal
Build a high-trust, SEO-driven affiliate site that helps users organize garages faster using clear comparisons, buying guides, and practical setup advice.

## Initial Scope
- Homepage
- Core guide pages (storage racks, bins, wall systems, tool organization)
- Comparison pages
- Trust/legal pages:
  - About
  - Contact
  - Privacy Policy
  - Affiliate Disclosure
- Technical baseline:
  - `robots.txt`
  - `sitemap.xml`
  - schema markup
  - internal linking

## Monetization
Amazon Associates compliant linking with project-specific tracking ID.

## Standards
- No fake reviews/ratings/testing claims
- No deceptive pricing language
- Buyer-oriented, useful, non-fluff content
- Mobile-friendly, fast-loading pages

## Status
Repository initialized. Build phase begins after tracking ID confirmation.
